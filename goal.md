# 프로젝트 계획서 (MVP 버전)

---

## 📌 프로젝트명

> **AI Office** 또는 **AI Companion**

---

## 🎯 목표

- 사용자의 대화, 목표, 프로젝트를 **장기적으로 기억**하고
- 대화 중 필요한 기억을 **꺼내와 활용**하며
- 필요한 경우 **먼저 질문을 던지는**
- 개인 맞춤형 AI 동료 구현

---

## 🔭 최종 비전

```
사용자
  ↓
AI 동료
  ├── 기억
  ├── 목표 추적
  ├── 프로젝트 관리
  ├── 문서 참조
  └── 능동 질문
  ↓
장기적인 관계 형성
```

---

## 🛠 기술 스택

| 영역   | 기술              | 비고                            |
| ------ | ----------------- | ------------------------------- |
| 프론트 | React, TypeScript |                                 |
| 백엔드 | Node.js, Express  |                                 |
| AI     | Ollama            | 초기 모델: Gemma 3 12B 또는 27B |
| DB     | SQLite            | 초기에는 벡터DB 사용 안함       |

---

## 🚀 개발 단계

### 1단계 — MVP

**목표:** AI와 채팅 가능

**구조:**

```
React → POST /chat → Node → Ollama → Response
```

**구현 기능:**

- 채팅
- 대화 저장
- 채팅 이력 조회

---

### 2단계 — Memory System

**목표:** AI가 사용자를 기억하기 시작

**테이블:** `users` / `messages` / `memories` / `goals`

**memory 예시:**

```json
{
  "type": "interest",
  "content": "사용자는 Node.js에 관심이 있음",
  "importance": 0.8
}
```

---

### 3단계 — Memory Extraction

**목표:** 대화에서 기억 자동 생성

**흐름:**

```
사용자: "나는 BMW F57을 타고 있어"
  ↓
Extractor: 장기 기억 여부 판단
  ↓
결과: { "save": true, "type": "profile", "content": "사용자는 BMW F57 보유" }
  ↓
저장: SQLite
```

---

### 4단계 — Memory Retrieval

**목표:** 질문에 맞는 기억 검색

**흐름:**

```
사용자: "요즘 뭘 공부해야 할까?"
  ↓
검색: Node.js / AI Agent / 취업 준비
  ↓
프롬프트 조립:
  시스템 프롬프트 + 사용자 프로필 + 관련 기억 + 현재 질문
```

---

### 5단계 — Goal Tracking

**목표:** 목표 중심 대화

**예시 목표:**

- Node.js 공부
- AI 동료 개발
- 취업 준비

**AI 역할:**

- 목표 생성 / 수정 / 완료 / 상태 관리

---

### 6단계 — Proactive Agent

**목표:** AI가 먼저 질문

**예시:**

```
Node.js 공부 시작
  ↓
7일간 관련 언급 없음
  ↓
AI: "Node.js 공부는 어떻게 되어가?"
```

## 💡 핵심 경쟁력

이 프로젝트의 경쟁력은 **모델 성능**이 아니다.

| 핵심 가치                   | 설명                             |
| --------------------------- | -------------------------------- |
| 🧠 나를 기억함              | 사용자의 정보와 맥락을 장기 보존 |
| 🎯 목표를 추적함            | 사용자의 목표 변화를 지속 관리   |
| 🙋 먼저 물어봄              | 능동적으로 대화를 시작           |
| 🔗 프로젝트를 이어서 이해함 | 이전 맥락을 끊김 없이 연결       |

> **첫 목표는 "최고의 AI 모델"이 아니라** > **"한 달 전 이야기를 이어서 할 수 있는 AI 동료"**
